# tiktok-publisher-legal

Legal documents for **OpenClaw Draft Kit** — a desktop productivity tool that helps content creators upload video drafts to their own TikTok drafts box via TikTok's official Content Posting API.

## Hosted documents

These documents are published via GitHub Pages and are referenced by the Application's TikTok for Developers submission:

- 📄 **Terms of Service**: https://aixier.github.io/tiktok-publisher-legal/terms
- 🔒 **Privacy Policy**: https://aixier.github.io/tiktok-publisher-legal/privacy
- 🏠 **Index**: https://aixier.github.io/tiktok-publisher-legal/
