## 1. Description of the Application

OpenClaw TikTok Publisher is a desktop productivity tool that helps individual content creators manage their own video drafts and upload them to their own TikTok drafts inbox via TikTok's official Content Posting API. The Application integrates with TikTok for Developers using Login Kit (OAuth 2.0) and the Content Posting API.

The Application's functionality is limited to:

- Authorizing the User's own TikTok account via OAuth 2.0
- Fetching basic profile information (open_id, display_name, avatar) to identify the connected account
- Uploading video files that the User has explicitly selected into the User's own TikTok drafts inbox
- Polling the status of uploads initiated by the User
- Listing the User's own published videos (if the corresponding scope is granted)

The Application **does not** publish content without the User's explicit confirmation inside the TikTok mobile application. All final publishing decisions remain with the User and occur within TikTok's official application.
